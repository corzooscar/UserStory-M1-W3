# 📣 UserStory-M1-W3 📣
## 📦 Inventory Manager 📦
A terminal-based inventory management system built in Python. Supports full CRUD operations, business statistics, and CSV persistence so your data survives between sessions.

### ► What it does
  1. Add, view, search, update, and delete products
  2. Calculate inventory statistics (total units, total value, most expensive product, highest stock)
  3. Save inventory to a CSV file
  4. Load inventory from a CSV file with merge or overwrite options
  5. Input validation and error handling throughout — the program never crashes on bad input
